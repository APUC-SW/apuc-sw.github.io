# 공식 프로젝트 위키에 오신 걸 환영합니다
이 페이지는 아직 작업 중입니다. 현재는 새로운 기능 테스트 실험이 진행 중입니다.<br>
귀하께서는 현재 게임/RP 문서 위키를 보고 계십니다.

## 개요
이 페이지는 메인 페이지입니다.<br>
이 문서는 가능한 경우 조만간 완성될 예정입니다.

## 기여
이 섹션은 위키 문서에 기여한 사람 및 위키 기여 가이드 정보만 다룹니다. 자체 위키 플랫폼 개발에 기여한 사람은 다루지 않습니다. 코드 기여자의 크레딧 정보는 해당하는 소스 코드 블럭에 주석으로 포함되었기 때문입니다.<br>
어쨌든, 아직 아무도 현재 이곳에 작성된 모든 문서에 기여하지 않았습니다.<br>
리포지토리 내용을 열람하거나 개발자 도구를 통해 소스 코드를 보는 것까지 다 막을 수는 없겠지만, 본 사이트의 소스 코드를 그런 식으로 열람하는 것은 리버스 엔지니어링으로 간주합니다. 본 소스코드의 일부 코드 블록이 MIT 라이선스일 가능성이 있지만 그렇다고 모든 소스 코드가 MIT 라이선스를 가지지는 않으므로 소스 코드의 열람 또는 복제를 일절 권장하지 않습니다. 귀하께서 본 사이트의 소스 코드를 그대로 재사용함으로써 발생하는 모든 라이선스 문제는 당사에서 일절 책임지지 않습니다. 소스 코드에 누군가의 이름과 출처 링크가 포함된 코드 스니펫이 있다면 해당 코드 블럭의 라이선스는 해당 저자의 것이며, 당사에서 소유권을 주장하지 않는 것입니다.

# FAQ
## 왜 사이트/프로젝트를 홍보하는거죠?
이곳은 프로젝트에 대한 문서를 통합 정리한 곳입니다. 여기에는 프로젝트에 대한 리디렉션 URL을 포함할 수는 있지만, 프로젝트에 대한 <b>그 어떠한 형태의 광고</b>도 포함되지 않습니다. 따라서 이 위키 자체는 순수하게 비상업적입니다.<br>
우리는 이용자들이 서비스 이용에 대한 도움을 받을 수 있도록 위키 문서의 리디렉션 링크를 제공할 수는 있지만, 저희 공식 위키 자체를 직접 홍보하지는 않습니다. 만약 어디선가 위키를 홍보하는 듯한 글을 보셨다면 <b>절대 그 링크를 클릭하지 마세요.</b>

## 각 문서의 리비전을 볼 수 있나요?
아니요, 이것은 단순히 개발자 문서로 활용되기 위한 소규모 위키에 불과합니다. 대 백과사전을 만들려는 것이 아니므로, 저장소의 Git 커밋 리비전 기록을 보기 좋게 불러오는 기능을 추가할 계획은 없습니다. 마치 제품 사용설명서가 모든 개정판 버전을 보여주지 않는 것과 같습니다.

## 당신의 사이트 보안 시스템은 너무 약해요!
네, 정확하게 보셨습니다. 왜냐하면 그건 진짜 보안 시스템이 아니기 때문입니다. 단지 저는 아는 사람만 아는 문제지를 여러분들께 보여주었을 뿐입니다. 딱히 보안 상 해커로부터 숨겨야해야 하는 콘텐츠가 없기 때문에 재미삼아 만든 것일 뿐이며, 진짜 민감한 정보는 리포지토리에 들어있지도 않습니다.<br>
이 글을 보고 계시다면 아마도 개발자 도구 등의 우회적인 방식으로 들어오신 분일 겁니다. 아니면 정말 운이 좋게 제 인증 서버가 정상 작동했거나요. 아마도 후자일 가능성은 거의 없을 겁니다. 왜냐하면 저의 외부 서버 의존성을 최소화하려는 노력 때문에 대부분의 서버사이드 기능은 제 사이트에서는 기본적으로 동작하지 않기 때문입니다.<br>
생각해보세요. 전세계 모든 GitHub Pages 기반 사이트에서 아무리 잘 만들어진 보안 시스템이라고 해봤자, 공개된 리포지토리 안에서는 모두 무용지물입니다. 이것은 결국 정적 사이트의 최대 단점이며, 쉽게 보완할 수 없습니다.
