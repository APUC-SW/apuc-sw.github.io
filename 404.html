ERROR 404 - NOT FOUND

The page could not be found or the URL address is invalid.


Self-Diagnosis:

  1. First, calm down and take deep breaths to regain your composure.
  2. Check if your device is connected to the internet.
  3. Refresh the page to see if this is a temporary issue.
  4. Check if the correct URL is entered in the address bar.
  5. Check if you are viewing an HTML document downloaded by mistake via 'Save this page'.
  6. Check if the GitHub server is undergoing maintenance.
  7. Check if a conflict with a browser extension is the cause.
  8. Check if your firewall, VPN, or proxy settings are the cause.
  9. Click the 'Commit History' button at the bottom to check for recent changes to the site.
  10. Check the news or weather forecast to see if a natural disaster is causing the internet to slow down.
  11. Check if your AI girlfriend/boyfriend has gone berserk and is interfering with your internet surfing.
  12. If you cannot identify any of the causes, worry that a doomsday scenario might already be unfolding in reality.

If you still cannot identify the cause despite diagnosing using the method above...

well, there is nothing we can do. Do your best, or just panic. This is something we did not foresee either.
